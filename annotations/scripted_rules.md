## Overview
The `scripted_rules/` folder exists as a load target but contains no vanilla definitions in v1.1.10 — only the placeholder `scripted_rules.info` file. The system is reserved for named boolean rule overrides that replace hardcoded engine checks, but Paradox has not populated it in this version.
