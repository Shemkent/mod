# EU5 Game File Structure Guide

This guide is the master index of identified game systems. Update it during Phase A (File Discovery) whenever new systems or keywords are found. Each entry links to the full annotation and lists the keywords Claude should use to recognize files belonging to that system.

## Entry format

Each system entry follows this structure:

```
## System Name
**Annotation:** [system_slug.md](system_slug.md)
**Folder:** `in_game/common/[folder]/`
**Keywords:** top_level_block_key, another_key, ...
**Stage:** stub | files-identified | in-progress | complete

Short description of what this system does.
```

`Keywords` are the top-level identifiers (block names, keys) found at the root level of the vanilla script files. Use these to recognize which system a file belongs to during Phase A.
