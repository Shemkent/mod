# Europa Universalis V Modding Project

Modding and annotation project for EU5 gameplay changes.

## Structure

- **`annotations/`** — Documentation of vanilla game file systems (for reference)
- **`mod/`** — Mod directories, each with `.metadata/` for mod annotations
- **`.gitignore`** — Git configuration for this project

## Mods

- **Better Piracy Workaround** — Reduces pirate durability to manageable levels

## Workflow

1. Use `annotations/` to understand game systems
2. Create mods in `mod/[mod_name]/` with standard EU5 structure
3. Store mod-specific annotations in `mod/[mod_name]/.metadata/`
4. Commit changes with descriptive messages
